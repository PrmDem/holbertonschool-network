# Network

As programmers, it is important to understand how and why our online applications or websites can be displayed and shared with the world. This Holberton School project is made to teach us the basic principles of networking â€“ and not the social kind.<br/>

## Requirements
### for Bash script files:
* Allowed editors: vi, vim, emacs
* All files will be interpreted on Ubuntu 22.04
* All files should start with `#!/usr/bin/env bash` and end with a new line
* The second line should be a comment explaining what is the script doing
* All files must be executable
* All scripts must pass shellcheck without any error
### for Answer files:
* For multiple choice question tasks, the number of the correct answer must be entered on a new line on the file
* If several questions are asked in a single task, all answers must be on the same file
